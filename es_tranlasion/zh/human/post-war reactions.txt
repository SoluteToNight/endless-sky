			label eyes
			`左边的双胞胎笑着说：“请原谅我们，如果我们睁大眼睛，相信我们自己的枪。”双胞胎都点点头，然后前往各自的船上。`
				decline
			label shrug
			`双胞胎耸耸肩，前往各自的船。`
				decline
			label scowl
			`双胞胎的脸都扭曲着，这可能表明他们感到尴尬或恼怒。这很难说，因为他们的面部毛发阻碍了任何情感力量。他们朝船的方向走去。`
				decline

	to offer
		has "main plot completed date"
		"days since start" - "main plot completed date" > 30
		random < 30
	on offer
		"war reactions" ++
		conversation
			`当你的船加油时，你正在咖啡馆里放松，两位中年船长来到你的餐桌旁。一个是海军，另一个是自由世界，它们彼此有着惊人的相似之处。`
			`“雷恩船长，R.N.S. Camuy，”其中一人说道。 “<ship> 怎么样，<last> 队长？”`
			`“菲奥娜队长，F.S. 猴面包树，”另一个说道。 “我们都看到你在与哈巴狗的战斗中。你是这场战争中最有成就的队长！`

						not "WR: Captains: navy"
				`“你发现新的外星人了吗？”`
					goto aliens
					to display
						not "WR: Captains: aliens"
						has "WR: Captains: navy"

					to display
						not "WR: Captains: sister"

					to display

						not "WR: Captains: opposite"

					decline

						"WR: Captains: question" <= 1

				decline

			action
				set "WR: Captains: navy"
			`“警方联合行动，”雷恩队长说道。 “这比你想象的更常见。”`
			`菲奥娜船长点点头。 “我们正在关闭一个银河蜘蛛芬克走私活动。这是一种非法致幻剂。不会让人上瘾，但仍然很危险。运营该活动的组织规模庞大，资金充足，装备精良。”`
			`“不过，这不是无畏舰和辅助舰的对手，”雷恩吹嘘道。`

			`“当然，我们还有另一个任务，”菲奥娜说。 “无论我们走到哪里，我们都会寻找新的外星人活动。”`

			label aliens
			action
				set "WR: Captains: aliens"
			`“只有常见的各种恶作剧，”雷恩说。两人都笑了。`
			`“不过，说实话，自由世界参议院和共和国议会都因害怕外星人而武装起来，”菲奥娜说。 Rain 补充道，“这一次只是象征性地举起了手臂。`
			`“如果我们发现新的外星人，我们会尽快通知必要的当局。你也应该这样做。保持谨慎，避免冲突，如果需要就寻求帮助。”`
				goto questions
			
			label sister
			action
				set "WR: Captains: sister"
			`“是的，”菲奥娜说，“作为我的妹妹并没有阻止我炸毁她的巡洋舰！不过，我们救出了她飞船的逃生舱；它们正在转向一颗星星，令人担忧。”`
			`Rain继续说道，“当我在姐姐的新无畏舰上醒来时，你能想象我的惊讶吗？”`

				goto questions
			
			label opposite
			action
				set "WR: Captains: opposite"
			`“嗯，雷恩几年前就去了地球加入海军学院，”菲奥娜上尉说。`
			`“在自由世界成立之前，菲奥娜留在了跳远并加入了当地民兵，”雷恩上尉说道。 “并不是我们想互相争斗，而是我们都相信自己的立场。”`
				goto questions
